# Strength Training, Progressive Overload

Notes from reading and experimenting on a PPL (Push / Pull / Legs) routine. Captured 2026-04-08.

## The Core Idea

Strength gains come from progressively increasing the demand on the muscle over time. The variable can be:
- Weight (most common)
- Reps (at the same weight)
- Sets (volume)
- Tempo (slower eccentrics)
- Range of motion

Pick one variable to push at a time. Don't change everything at once.

## Linear Progression (where I am)

Add weight every session if all sets at the prescribed reps are completed cleanly. For the main lifts:
- Bench / OHP: +5lb when 5x5 is clean
- Squat / Deadlift: +10lb when 5x5 is clean
- Cable / accessory work: +5lb or 1-2 reps when smooth

When linear progression stops working (usually after 3-6 months for a non-beginner), switch to:
- Double progression (rep range, then weight)
- Periodization (cycles of intensity vs volume)

## Practical Rules I'm Following

1. **Form before weight.** If form breaks down at the last rep, the weight is too high.
2. **Track everything.** Without a log, "progress" is vibes. The reading tracker pattern works here too. One markdown table per program.
3. **Deload every 4-6 weeks.** A deload week = same exercises, lighter weight. Resets fatigue.
4. **Sleep beats training.** Bad sleep weeks: maintain, don't push.

## Current Program (PPL, 3x/week)
